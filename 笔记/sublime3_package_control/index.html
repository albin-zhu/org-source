Sublime Text的package-control安装

笔记
  • package-control
  • sublime

Package Control

记录一下方便以后查找，以下翻译自外文。

安装

命令行自动安装

安装的最简单的方法是通过sublime的文本控制台。控制台通过访问CTRL +` 快捷方式或 View > Show Console。一旦打开，粘贴以下代码

sublime text 3

sublime text 2

此代码创建安装的软件包文件夹，你（如果需要的话），然后下载 Package Control.sublime包进去。下载将通过HTTP，而不是HTTPS进行，由于Python标准库的限制，但是该文件将使用SHA-256验证。

手动安装

如果由于某种原因控制台不能安装（墙内），请执行以下步骤手动安装程序包控制：

点击 Preferences > Browse Packages … 菜单
浏览了一个文件夹，然后进入安装的软件包/ 文件夹
下载包Control.sublime包 ，并将其复制到安装的软件包/目录
重新Sublime Text.
